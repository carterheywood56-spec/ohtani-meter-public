指標の考え方

「なぜこの測り方をしているのか」「どこまで信頼してよいか」「データの出所」を説明します。2スコアの定義や基本的な読み方は About ページをご覧ください。

A · Why Two Scores

なぜ結果と内容を分けるのか

打撃の評価によく使われる指標（打率・OPS・wOBAなど）は、打席の最終結果を集計したものです。シーズン全体を通じて見るときには信頼できますが、数週間〜1か月程度の短い期間では「運」の影響が大きく、実際の打撃の質と乖離することがあります。

たとえば同じ「打率2割台」でも、鋭い打球を連発しながら野手の正面を突いている月と、スイングが崩れていて当てるだけになっている月とでは、その後の展開が異なります。結果指標だけを見ていると、この違いが見えません。

スイング内容スコアは、ヒットになったかどうかに依存しない指標で構成することで、結果の影響を切り離しています。2つのスコアのズレを見ることが、このツールの主な使い方です。

具体例 · Same Average, Different Swings

同じ打率.250でも、内容と結果のスコアはこう違う

これは概念図ですが、月単位で「同じ打率」が観察された場合に、内容スコアと結果スコアの示唆は大きく分かれます。

A · 鋭い打球が野手の正面

チェイス率 0.21 / hard_hit% 0.62。ボール球を振らず、振った球は速い。野手の正面を突いて打率は伸びないが、打球の質は高い。

内容 90 · 結果 74（差 -16）
B · 当てるだけ、本塁打で結果を支える

チェイス率 0.34 / hard_hit% 0.50。ボール球に手を出し、当てるだけの打撃。本塁打が出続ける限り wOBA は維持される。

内容 65 · 結果 80（差 +15）

結果指標だけ見れば「同じ.250」でも、A はその後の反発が期待でき、B は崩れの初期サインです。

B · The Six Inputs

なぜこの6要素なのか

スイング内容スコアは現在、チェイス率・Z-Swing%・コンタクト率・hard_hit%・SweetSpot%・Barrel%の6指標を組み合わせています。選定の基準は「打席の最終結果に依存せず、スイングの質を直接反映するか」です。

チェイス率・Z-Swing%・コンタクト率は、選球眼とスイング判断・ミートの精度を表します。ヒットが出なかった打席でも計測でき、状態の変化を早めに拾いやすい指標です。

hard_hit%（フェアボールのうち打球速度95mph以上の割合）は、コンタクト率を補完するために加えています。コンタクト率が同じでも、強い打球の割合が低ければ「当てただけ」の状態である可能性が高く、その差を区別するためです。母数はフェアボール（インプレー打球）に限定しています。

SweetSpot%・Barrel%は打球品質を多角的に捉えるために加えています。SweetSpot%は打球角度8〜32°の理想帯に入った割合、Barrel%は打球速度と角度の組み合わせが最も長打になりやすいゾーンに入った割合です。hard_hit%と同じくフェアボール限定で計算しています。

逆に使っていないのは、「ヒットになったか」「塁に出られたか」のような最終結果です。それらは内容ではなく運の要素を多く含み、内容スコアに混ぜると目的が崩れるためです。

具体例 · What Each Input Captures

6指標が反映する打席内容

各指標は「打席のどの局面のどの質」を捉えているか。値の方向（高い／低いどちらが良いか）も整理しておきます。

指標 値が高いとき（or 低いとき）の打席イメージ
チェイス率
低いほど良い		 0.45 → ボール球を見極められず、追って空振り・凡打。
0.20 → ストライクゾーン外をきっちり見送る。
Z-Swing%
高いほど良い		 0.75 → ストライクをしっかり振りに行ける。
0.55 → ストライクを見逃して追い込まれる。
コンタクト率
高いほど良い		 0.80 → 振った球にしっかりバットが当たる。
0.60 → 振っても空振り、三振が増える。
hard_hit%
高いほど良い		 0.65 → フェアボールの大半が95mph以上の鋭い打球。
0.40 → 詰まったゴロや弱いポップフライが目立つ。
SweetSpot%
高いほど良い		 0.45 → 角度8〜32°の理想帯に多く打球が乗る。
0.25 → 浮きすぎ（フライ）or 叩きつけ（ゴロ）が増える。
Barrel%
高いほど良い		 0.30 → 速度＋角度ともに長打ゾーン、本塁打が出やすい。
0.10 → 速いけど角度が悪い／角度はいいけど打球速度が出ない。

各指標は単独で「打撃の良し悪し」を決めるものではなく、6つの組み合わせで打席の質を立体的に捉えるための材料です。

C · Self-Reference Baseline

なぜ本人基準なのか

スコアは0〜100に変換していますが、その基準は大谷本人の2022〜2025年の150球ローリング窓です。「リーグ平均と比べていくつ」ではなく、「大谷の過去の状態と比べてどの位置か」を示しています。

直近150球ローリング窓のイメージ

各丸は試合1試合分。試合が増えるたびに窓が右へスライドし、その時点のスコアが更新される。

この設計には理由があります。大谷は打撃の水準がリーグ内でも高い選手です。リーグ平均を基準にすると「常に高スコア」になり、状態の変化が見えにくくなります。本人の過去データを基準にすることで、「大谷らしい状態かどうか」の変化を敏感に拾うことができます。

ただし、これは他の選手との比較には使えない設計です。「80点」が大谷の平常水準であり、絶対的な良し悪しを示す数字ではありません。

D · Reliability

どこまで信頼してよいか

このツールが示せることと、示せないことをはっきりさせておきます。

✓ 信頼してよい使い方

信頼してよい使い方

  • 「今の内容スコアは、過去の大谷と比べてどの位置か」を参照する
  • 結果スコアと内容スコアのズレを確認し、「慌てるべき状態かどうか」の判断材料にする
  • 数か月にわたるトレンドの変化を観察する
—

信頼しすぎてはいけない使い方

  • シーズン序盤（10〜20試合程度）の数値の断定的な読み取り。サンプルが少ない時期は数値が大きく振れやすく、傾向の判断に向きません。50試合以上のデータが蓄積されると安定度が増します。
  • 「内容が良いから必ず回復する」という予測。内容スコアと結果スコアのズレが縮まるかどうかには個別の事情が影響します。
  • このスコアに含まれない要素の見落とし。球種・コース別の対応、相手投手の質、身体コンディションなどは反映されていません。
このツールは参考指標です。確定値ではありません。 個人が趣味で作ったものであり、公式統計ではありません。誤りや見落としがある可能性があります。
E · Data Source

データの出所

Statcast（MLB公式の計測データ）を使用しています。球速・打球速度・スイングの軌道・ゾーン情報など、1球ごとの詳細データが含まれています。
